🔹 Project Name

 Digital Tools Store

🔹 Description

A website where users can purchase digital tools like software, templates, or subscriptions. It includes a product list, cart functionality, and a fully responsive design.

🔹 Technologies Used
React.js
Tailwind CSS
DaisyUI
React-Toastify
JSON (for product data)
🔹 Features
Responsive Design – Looks great on all devices.
Cart Functionality – Add, remove, and checkout products easily.
Product Cards – Display products in a 3-column layout with name, price, features, and buttons.
